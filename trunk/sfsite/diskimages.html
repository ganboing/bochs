Disk Images with Pre-installed Systems
    We are trying to collect a minimal hard disk or floppy disk image for all free supported operating systems that Bochs can run. Please don't use the System and VGA Bioses contained in the tarballs, as they are outdated. More info...

WARNING: Some disk images have not been updated recently, so some Bochsrc hacking might be required.

Other disk images are available from the file section of the Bochs pages on SourceForge

  • FreeDOS
    5-meg hard disk image which boots into FreeDOS.
  • Linux
    4-meg hard disk image which boots into Linux. It should boot ok.
  • DLX Linux
    A 10-meg disk that boots DLX Linux (1.3.x kernel).
  • NetBSD
    Bootable floppy image. Fails in Bochs under bugfix2 with hard disk panic.
  • OpenBSD
    Bootable floppy image. Fails in Bochs under bugfix2 with hard disk panic.
  • PicoBSD
    Floppy image that boots FreeBSD, not a complete installation.
  • Pragma Linux
    Floppy image that boots Pragma Linux.
  • Debian images, contributed by Robert Millan
    • Debian 3.0r0 Image (77MB download)
    • Debian 2.2r5 Image (32MB download, 505MB uncompressed) and associated bochsrc

Other Images
  • Linux kernel image for x86-64 from Andi Kleen at SuSE, which can be used as a boot floppy. Here is the System.map file for it, which can be useful for debugging.
  • BOXCARZF.ZIP
    DOS executable from Leyland Needham (suboner@aol.com) that may expose a Bochs FPU bug
